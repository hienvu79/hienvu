# Hello Word!
## Cách để chuyển đổi tất cả các ký tự trong một chuỗi về dạng chữ in hoa.
```
<body>
<p id="demo"></p>
<p id="demo1"></p>
</body>
```
```
<script>
var a = "Hello Word";
var b = a.toUpperCase();
document.getElementById("demo").innerHTML = a;
document.getElementById("demo1").innerHTML = b;
</script>
```
**Ví dụ**
```
Hello Word
HELLO WORD
```
